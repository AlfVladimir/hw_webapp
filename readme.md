БД postgres или sqlite (параметры подключения после импорта в файле).
При запуске создает таблицы (если не существуют).
Приложил файл insert.sql, в котором тестовые данные.

Запуск через uvicorn:
uvicorn webapp:app

Эндпоинты по задаче как в описании:
GET
/stores/
Все магазины

/items/
Товары

/items/top/
Топ 10 товаров за месяц по количеству

/stores/top/
Топ 10 магазинов по сумме проданного

POST
/sales/
На входе item_id, store_id
Если кривой json или таких товара/магазина не существует в базе возвращает 400.
